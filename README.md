# Food Delivery App UI Design

This is a mobile UI/UX prototype for a food delivery app created in Figma.

## Features
- User Login/Signup
- Menu Browsing
- Cart & Payment Screens
- Order Tracking Flow

## Tools Used
- Figma
- Auto Layout, Prototyping, Smart Animate
